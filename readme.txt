Voici le dashboard interactif du projet "Implémentez un modèle de scoring" du parcours Data Scientist.


Requirements :
- pandas 	https://pandas.pydata.org/pandas-docs/stable/getting_started/install.html
- dash 		https://dash.plot.ly/installation
- plotly 	https://plot.ly/python/getting-started/
- pickle 	https://machinelearningmastery.com/save-load-machine-learning-models-python-scikit-learn/


Détails des fichiers :
- app.py 	script de lancement du dashboard
- assets	contient les logo/images et .css du dashboard
- data		contient les fichiers sources - à télécharger: https://drive.google.com/open?id=1skfd9vA2rFFXp6br42nKSZOs_u4w_67M
